MongoDB-聚合框架

xucongcong
2022-10-07
  • MongoDB

MongoDB的聚合框架是一个计算框架，其可以作用在一个多个集合上，对集合中的数据进行分析、统计、复杂数据运算。

MongoDB的聚合运算过程为一个Pipeline，其由多个Stage组成，每个Pipeline接收一系列文档作为原始数据，通过每个Stage的计算传递中间结果，最后输出最终结果。

# 1. 聚合运算的语法格式

pipeline = [$stage1, $stage2, $stage3...]
db.collection_name.aggregate(pipeline, {options})

常用stage：

步骤 作用 SQL对比
$match 过滤 WHERE
$project 投影 AS
$sort 排序 ORDER BY
$group 分组 GROUP BY
$skip/$limit 结果限定 SKIP/LIMIT
$lookup 左连接 LEFT JOIN
$unwind 展开数组
$facet/$bucket 分面搜索
$graphLookUp 图搜索

每个步骤中还可使用相应的运算符：

$match：$eq、$gt、$gte、$lt、$lte、$and、$or、$not、$in …

$project：$map、$reduce、$filter、$range …

$group：$sum、$avg、$push、$addToSet、$first、$last、$max、$min …

# 2. 聚合运算的使用场景

“OLTP(on-line transactionprocessing)-> 联机事务处理。 OLAP(On-Line Analytical Processing) -> 联机分析处理。 从字面上来看OLTP是做事务处理,OLAP是做分析处理。从对数据库操作来看,OLTP主要是对数据的增删改,OLAP是对数据的查询。

聚合运算在oltp中可用于分页查询、投影等计算，也可在oltp中做报表统计，如：

  • 分析一段时间内某字段的总值、平均值
  • 计算一段时间内的净利润
  • 分析年龄分布、成绩分布
  • 统计绩效

# 3. 聚合查询的示例

示例一：

# -------SQL--------

SELECT 
	FIRST_NAME AS '名'
	LAST_NAME AS '姓'
FROM Users
WHERE GENDER = '男'
SKIP 100
LIMIT 20

# -------MQL--------

db.users.aggregate([
	{$match: {gender: "男"}},
	{$skip: 100},
	{$limit: 20},
	{$project: {
		'名': '$first_name',
		'姓': '$last_name'
	}}
])

示例二：

# -------SQL--------

SELECT
	DEPARTMENT, 
	COUNTER(NULL) AS EMP_QTY
FROM Users
WHERE GENDER = '女'
GROUP BY DEPARTMENT HAVING COUNT(*)<10

# -------MQL--------

db.users.aggregate([
	{$match: {gender: '女'}},
	{$group: {
		_id: '$DEPARTMENT', # 按照指定字段分组，可以指定null不分组即所有记录作为一个组
		emp_qty: {$sum: 1}
	}},
	{$match: {emp_qty: {$lt:10}}}
])

示例三：特有的$unwind展开操作

> ddb.students.findOne()

{
	name: "foo",
	score: [
		{ subject: "math", score: 100 },
		{ subject: "sport", score: 60 },
		{ subject: "english", score: 100 },
	]
}

> db.students.aggregate([{ "$unwind": "$score"}])
{ name: "foo", score: { subject: "math", score: 100 } }
{ name: "foo", score: { subject: "sport", score: 60 } }
{ name: "foo", score: { subject: "english", score: 100 } }

示例四：使用特有的$bucket统计不同价格区间（0～10、10～20、其他）的商品数

db.products.aggregate([{
	$bucket: {
		groupBy: "$price",
		boundaries: [0, 10, 20]
		default: "Other",
		output: { "count": {$sum: 1} }
	}
}])

示例五：使用特有的$facet 统计多个字段的分布（price、year的分布）

db.products.aggregate([{
	$facet: {
		price: {
			$bucket: {...}
		},
		year: {
			$bucket: {...}
		}
	}

}])
