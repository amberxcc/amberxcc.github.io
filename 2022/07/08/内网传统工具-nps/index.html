内网穿透工具-nps

xucongcong
2022-07-08
  • 计算机网络

nps是一款轻量级、高性能、功能强大的内网穿透代理服务器。目前支持tcp、udp流量转发，可支持任何tcp、udp上层协议（访问内网网站、本地支付接口调试、ssh访问、远程桌面，内网dns解析等等……），此外还支持内网http代理、内网socks5代理、p2p等，并带有功能强大的web管理端。


# 1. 应用场景

  1. 做微信公众号开发、小程序开发等----> 域名代理模式
  2. 想在外网通过ssh连接内网的机器，做云服务器到内网服务器端口的映射，----> tcp代理模式
  3. 在非内网环境下使用内网dns，或者需要通过udp访问内网机器等----> udp代理模式
  4. 在外网使用HTTP代理访问内网站点----> http代理模式
  5. 搭建一个内网穿透ss，在外网如同使用内网vpn一样访问内网资源或者设备----> socks5代理模式

# 2. 准备工作

  • 一台带有公网IP的云服务器（运行nps服务端）
  • 使用内网IP的服务器（运行nps客户端）

# 3. 启动nps服务端（Docker安装）

  • docker pull ffdfgdfg/nps

  • 下载conf文件 (opens new window)夹并解压，或前往项目主页 (opens new window)自行下载

  • 阅读文档 (opens new window)，根据需求修改配置文件（可选）

  • 启动：docker run -d --name nps --net=host -v <本机conf目录>:/conf ffdfgdfg/nps

启动后，使用docker logs nps 检查下是否有报错（端口冲突之类的）

# 5. 启动npc客户端（Docker安装）

  • docker pull ffdfgdfg/npc

  • 下载conf文件夹并解压，或前往项目主页自行下载

  • 阅读文档, 自定义按照不同的启动方式启动（可选）

  • 启动docker：

# 不指定npc配置
docker run -d --name npc --net=host ffdfgdfg/npc -server=<ip:port> -vkey=<web界面中显示的密钥> <other...>
# 手动指定npc配置
docker run -d --name npc --net=host -v <本机conf目录>:/conf ffdfgdfg/npc -config=/conf/npc.conf
